# Reports

Reports provide executive visibility into EOS health across teams and quarters.

## Reports home

**Path:** `/org/[slug]/reports`

### Summary KPIs

| Metric | Description |
|--------|-------------|
| Active rocks | Rocks in progress this quarter |
| Rock completion % | Done vs total |
| Open issues | Unresolved issues |
| Issues solved (7d) | Recent closure rate |
| Open to-dos | Outstanding 7-day tasks |
| Metric count | Scorecard measurables tracked |

### Team breakdown

Table of per-team health with links into team workspaces.

### Executive quarter drill-down

Select a quarter to see:

- Rock completion by team
- Issue trends
- Scorecard rollup (green/yellow/red)
- Cascade acknowledgment completion

### CSV export

Export report data for spreadsheets and board decks.

---

## Org scorecard rollup

**Path:** `/org/[slug]/scorecard`

Cross-team metric health summary (also accessible directly, not only from Reports).

---

## Activity vs audit

| Page | Audience | Content |
|------|----------|---------|
| **Activity** (`/activity`) | All members | Broad change feed with filters |
| **Audit log** (`/settings/audit`) | Owner, admin | Security and admin events (last 100) |

Use Reports for leadership rhythm; Activity for day-to-day tracing.

---

## Best practices

- Review Reports in monthly and quarterly leadership meetings.
- Export CSV before quarterly planning sessions.
- Track cascade completion to ensure messages landed.

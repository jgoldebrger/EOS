# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+K** / **Cmd+K** | Open global search |
| **Esc** | Close search, dialogs, or sheets |

## Search tips

After opening search with Ctrl+K / Cmd+K:

- Type a page name (for example `L10`, `V/TO`, `Reports`) to jump directly.
- Type a project name or transport load ID to open detail pages.
- Use arrow keys to highlight results and **Enter** to navigate.

## In L10 meetings

- Use the agenda sidebar to jump between sections (facilitator controls the active section for all participants).
- Section timer shows remaining time; overtime is highlighted when time expires.

## Browser tips

- Bookmark your team L10 hub for weekly access.
- Use multiple tabs for long meetings (scorecard in one section, issues in another) — changes sync in real time.
